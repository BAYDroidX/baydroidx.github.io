Get started

  • Device minimum android 16 (kitkat)
  • Root or NO Root device
  • Install latest version
  • Does not work on virtual app

How to use

  • Make sure the Free Fire ori & latest version
  • Then open the FFX application
  • If the app requests storage permission, grant it
  • To activate, click the button of the feature to be selected,select feature on dialog and click OK
  • To deactivate, click the feature button that has been selected again,select OFF on the dialog and click OK
  • And open the free fire application

Fix bugs

If you use the skin pack feature, it usually gets stuck loading in the free fire application or skin feature not working,How to fix stuck loading?

The problem is that you haven't downloaded the expansion packs yet, so you just need to download all expansion packs first.
